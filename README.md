# Estudo-de-curvas-de-luz-utilizando-LightKurve
  Este material tem como objetivo servir de guia para a utilização da biblioteca Lightkurve no estudo de curvas de luz. Ao longo do conteúdo, será mostrado como empregar as ferramentas oferecidas pela biblioteca para a análise e manipulação de dados observacionais, com ênfase na produção de histogramas.
